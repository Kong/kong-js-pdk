Plugin server and PDK (Plugin Development Kit) for Javascript language support in Kong.

Requires Kong >= 2.3.0.

## Documentation

See in [Kong Docs](https://docs.konghq.com/gateway-oss/latest/external-plugins/#developing-javascript-plugins).
